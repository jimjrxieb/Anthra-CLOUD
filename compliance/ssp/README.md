# System Security Plan (SSP)

FedRAMP Low baseline System Security Plan for the DVWA Compliance Demo.

## Documents

| File | Description |
|------|-------------|
| `ssp-skeleton.md` | Full SSP template with FedRAMP Low controls |

- **15 controls fully documented** in `../control-families/`
- Remaining controls have skeleton entries in the SSP template
